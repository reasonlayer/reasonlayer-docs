Reasonlayer logs every agent action with full execution context. You get audit trails and token usage metrics so you can monitor what your agents are doing and troubleshoot issues.

## What gets logged

- **Agent actions** — Every action your agent takes is recorded
- **Execution context** — Full context around each execution for debugging
- **Audit trails** — Complete history for compliance and security review
- **Token usage** — Track AI model consumption and costs

## Accessing logs

View logs and metrics in your Reasonlayer dashboard to monitor agent behavior and performance.
