The  supported chat channels for OpenClaw are **Whatsapp, Telegram, Discord, Slack, Google Chat, Mattermost, Signal, iMessage (via Bluebubbles), Microsoft Teams, Twitch, IRC, Feishu, LINE, NextCloud Talk, Matrix, Nostr, Tlon, Zalo, Zalo Personal, Webchat.**

For more information the OpenClaw docs can be found here: https://docs.openclaw.ai/channels

The easiest to configure is Telegram if you want to just get up and running. Here are some guides on setting up Telegram.
